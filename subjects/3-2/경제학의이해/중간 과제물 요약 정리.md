# 경제학의 이해 - 중간 과제물 요약 정리

## 과제 개요

### 과제 목적
- 학습 내용 및 읽기 자료의 이해도 확인
- 동영상 강의 및 교재 1-3강 범위

### 제출 기한
- **정시 제출**: 10월 10일(금) 9시 - 10월 20일(월) 18시
- **추가 제출**: 10월 21일(화) 9시 - 10월 23일(목) 18시 (자동 감점 -1점)
- 2차 추가 제출 기간 없음

### 제출 방법
- Myknou 학사정보 → 과제물 → 중간 과제물 제출
- 파일 형식: `.hwp`, `.hwpx`, `.doc`, `.docx`, `.pdf`
- 제공된 양식 파일 사용 (이름, 학번, 답만 작성)

## 작성 시 주의사항

### 기본 사항
- 서론-본론-결론 형식 사용 금지
- 학교 양식 표지 사용 금지
- 답에 대한 설명 불필요 (답만 작성)
- 모든 세부 질문에 답 필수

### 0점 처리 사항
- 다른 교과목 과제물 제출
- 과제물 내용이 없는 경우
- 파일이 열리지 않거나 내용이 깨진 경우
- 파일 보안 설정으로 열 수 없는 경우

## 평가 기준

### 채점 방식
- 복수 답 문제: 정답 모두 선택 시만 배점 획득 (부분 점수 없음)
- 질문 또는 보기에 오류가 있다고 생각하면 상세히 기술 가능
- 평가는 동영상 강의, 교재, 프리젠테이션 파일 기준

### 표절 관련
- 상업용 자료 또는 재학생 간 상호 표절 시 해당 답안 0점 처리
- 표절 검사 시스템 + 직접 비교 검토 병행
- 문장/문단 단위 표절도 감점 대상
- 일부 단어만 바꾸는 것으로 표절 회피 불가

## 인공지능 활용

### 사용 가능
- ChatGPT 등 생성형 AI 사용 가능
- 단, 설문 작성 필수

### 주의사항
- 생성형 AI 답변을 반드시 한 줄 한 줄 검증 필요
- 부정확하거나 논리적 오류 있는 경우 많음
- 긴 설명이지만 핵심 내용이 없을 수 있음
- 스스로 답을 먼저 작성 후 AI로 확인하는 방식 권장

## 문제 구성

### 진위형 (1-6번): 각 2점, 총 12점
주요 개념 확인 문제
- 경제의 정의
- 희소성의 개념
- 매몰비용과 의사결정
- 수요법칙과 공급법칙
- 시장의 자기조정 메커니즘

### 선택형 (7-15번): 각 2점, 총 18점
말차(Matcha) 시장 사례 분석
- 소셜 미디어 영향으로 인한 수요 변화
- 공급 제약 상황
- 수요-공급 모형을 통한 균형점 변화 분석
- 단기/장기 공급 변화
- 수요탄력성과 수입 변화
- 대체재 등장의 영향

## 생성형 AI 설문 (필수)
