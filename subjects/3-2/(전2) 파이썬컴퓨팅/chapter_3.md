# 01. 파이썬컴퓨팅: 변수와 데이터 타입의 기초

## 학습 목표
1. 파이썬 변수의 작동 원리가 컴퓨터 메모리 할당과 어떻게 연결되는지 설명할 수 있다.
2. 숫자형(정수, 실수)과 문자열형 데이터 타입의 차이점을 이해하고 올바르게 사용할 수 있다.
3. 할당문(`=`)을 사용하여 변수에 값을 저장하고, `print()` 함수로 변수의 값을 출력할 수 있다.

---

## 📝 요약 (Summary)
1.  **변수(Variable)와 메모리**: 변수는 데이터를 저장하기 위해 이름 붙인 메모리 공간입니다. 파이썬에서 `x = 10`과 같이 변수에 값을 할당하면, 데이터 `10`이 메모리에 저장되고 `x`라는 이름이 그 메모리 주소를 가리킵니다. 이를 통해 데이터를 재사용하고 관리할 수 있습니다.
2.  **데이터 타입(Data Type)**: 변수가 저장할 수 있는 데이터의 종류를 의미합니다. 파이썬은 **동적 타이핑(Dynamic Typing)**을 지원하여 변수 선언 시 타입을 명시할 필요가 없습니다.
    -   **숫자형**: `정수(int)`와 `실수(float)`로 나뉩니다. (예: `age = 25`, `pi = 3.14`)
    -   **문자열형(str)**: 작은따옴표(`'`)나 큰따옴표(`"`)로 묶어 텍스트 데이터를 표현합니다. (예: `name = "홍길동"`)
3.  **할당문과 출력문**: **할당문**은 등호(`=`)를 사용하여 변수에 값을 저장하는 명령입니다. **`print()` 함수**는 변수나 값의 내용을 화면에 출력하는 가장 기본적인 기능으로, 프로그래밍 결과를 확인하는 데 필수적입니다.

## ✏️ 심화 학습 (Study Subject)
1.  **가상 시나리오: 변수는 '상자'가 아닌 '이름표'**
    - **상황**: 간단한 카페 주문 프로그램을 만든다고 가정해 봅시다. `order_count = 10` 코드는 'order_count'라는 상자에 숫자 10을 넣는 것이라고 흔히 오해합니다. 하지만 파이썬의 동작 방식은 다릅니다.
    - **파이썬의 실제 동작**:
        1.  메모리의 한 공간에 숫자 객체 `10`을 만듭니다.
        2.  `order_count`라는 **이름표(label)**를 만들어 이 `10` 객체에 붙입니다.
        3.  이후 `order_count = 11` 코드를 실행하면, 상자 속 내용물을 바꾸는 것이 아니라 메모리에 새로운 숫자 객체 `11`을 만들고, `order_count` 이름표를 `10`에서 떼어 `11`에 새로 붙입니다. 더 이상 아무도 참조하지 않는 `10` 객체는 메모리에서 사라집니다(Garbage Collection).
    - **AI 학습 포인트**: 이 '이름표' 모델은 파이썬의 변수 할당과 메모리 관리 방식을 이해하는 핵심입니다. AI 학습 도우미에게 "만약 `a = 10` 다음에 `b = a`를 실행하고, 그 후 `a = 20`으로 바꾸면 `b`의 값은 왜 10으로 유지될까요?"라고 질문하여 '불변(immutable) 객체'의 값 복사 개념을 탐색하게 할 수 있습니다.

2.  **관점별 비교: 정적 타이핑 vs 동적 타이핑**
    - 프로그래밍 언어는 변수의 데이터 타입을 결정하는 방식에 따라 크게 두 가지로 나뉩니다.
    - **정적 타이핑 (Static Typing, 예: Java, C++)**: 변수를 선언할 때 반드시 데이터 타입을 명시해야 합니다. `int age = 25;` 처럼 `age`는 오직 정수만 담을 수 있습니다.
        - **장점**: 컴파일 시점에 타입 오류를 잡을 수 있어 안정성이 높고, 코드가 명확하며, 실행 속도가 빠를 수 있습니다.
        - **단점**: 코드가 길어지고, 유연성이 떨어집니다.
    - **동적 타이핑 (Dynamic Typing, 예: Python, JavaScript)**: 변수 선언 시 타입을 지정할 필요가 없습니다. `age = 25` 이후에 `age = "스물다섯"` 처럼 다른 타입의 값을 할당할 수 있습니다.
        - **장점**: 코드가 간결하고 유연하여 빠르게 개발할 수 있습니다.
        - **단점**: 실행 중에 예상치 못한 타입 오류가 발생할 수 있으며, 코드의 명확성이 떨어질 수 있습니다.

3.  **오개념 분석: 파이썬 초보자의 흔한 실수**
    - **오개념 1**: 할당문(`=`)은 수학의 '같다'와 동일하다.
        - **분석**: 프로그래밍에서 `=`는 '오른쪽의 값을 왼쪽의 변수에 할당(저장)하라'는 **명령**입니다. `x = x + 1` 이 수학적으로는 말이 안 되지만, 프로그래밍에서는 '기존 x 값에 1을 더한 결과를 다시 x에 저장하라'는 의미로 완벽하게 유효합니다.
    - **오개념 2**: 변수 이름은 마음대로 지어도 된다.
        - **분석**: 변수 이름은 규칙을 따라야 합니다. 숫자로 시작할 수 없고(`1st_player` (X)), 하이픈(`-`) 같은 특수문자를 포함할 수 없으며(`my-var` (X)), `if`, `for` 같은 **예약어(keyword)**는 사용할 수 없습니다. `my_player`, `myPlayer` 처럼 의미를 명확히 알 수 있는 이름을 사용하는 것이 좋습니다.

### 📝 문제

**1. [다음 코드 실행 후 `x`의 최종 값으로 올바른 것은?](#prob-1)**
   > ```python
   > x = 10
   > x = x + 5
   > ```
   > ① 10
   > ② 15
   > ③ 5
   > ④ 오류가 발생한다.

<br>

**2. 다음 중 파이썬 변수 이름으로 사용할 수 있는 것은?**
   > ① 1st_variable
   > ② my-variable
   > ③ my_variable
   > ④ for

<br>

**3. 다음 코드의 실행 결과로 올바른 것은?**
   > ```python
   > name = "Alice"
   > age = 25
   > print(name, age)
   > ```
   > ① "Alice" 25
   > ② Alice 25
   > ③ Alice25
   > ④ ("Alice", 25)

### 🧐 해설

<a id="prob-1"></a>
**1. 다음 코드 실행 후 `x`의 최종 값으로 올바른 것은?**
> **정답: ② 15**
> **해설**: `x = 10`으로 변수 `x`에 10이 할당됩니다. 다음 줄 `x = x + 5`는 현재 `x`의 값인 10에 5를 더한 결과(15)를 다시 `x`에 할당하라는 의미입니다. 따라서 `x`의 최종 값은 15가 됩니다.

<a id="prob-2"></a>
**2. 다음 중 파이썬 변수 이름으로 사용할 수 있는 것은?**
> **정답: ③ my_variable**
> **해설**: 파이썬 변수 이름은 영문자 또는 밑줄(`_`)로 시작해야 하며, 숫자, 영문자, 밑줄만 포함할 수 있습니다.
> - ① `1st_variable`: 숫자로 시작할 수 없으므로 잘못된 이름입니다.
> - ② `my-variable`: 하이픈(`-`)은 변수 이름에 사용할 수 없는 특수 문자입니다.
> - ④ `for`: 파이썬의 **예약어(keyword)**이므로 변수 이름으로 사용할 수 없습니다.

<a id="prob-3"></a>
**3. 다음 코드의 실행 결과로 올바른 것은?**
> **정답: ② Alice 25**
> **해설**: `print()` 함수에 여러 개의 인자를 쉼표(`,`)로 구분하여 전달하면, 각 인자의 값을 공백으로 구분하여 한 줄에 출력합니다. 따라서 문자열 변수 `name`의 값인 `Alice`와 숫자 변수 `age`의 값인 `25`가 공백으로 띄워져 `Alice 25`와 같이 출력됩니다.

## ✅ O/X 확인문제(연습문항 개수의 2배)

**1.** [파이썬 변수 이름은 숫자로 시작할 수 있다.](#ox-1) **(O / X)**
**2.** [파이썬에서는 변수를 선언할 때 반드시 `int`나 `str`과 같은 데이터 타입을 명시해야 한다.](#ox-2) **(O / X)**
**3.** [코드 `x = 10`에서 등호(`=`)는 'x와 10이 같다'는 수학적 의미를 가진다.](#ox-3) **(O / X)**
**4.** [파이썬에서 문자열은 작은따옴표(`'`)와 큰따옴표(`"`)를 모두 사용하여 만들 수 있다.](#ox-4) **(O / X)**
**5.** [`print("Hello", "World")` 코드는 "HelloWorld"라고 공백 없이 출력된다.](#ox-5) **(O / X)**
**6.** [`a = 100` 이고 `b = 100` 일 때, `a`와 `b`는 메모리에서 같은 숫자 100 객체를 가리킬 가능성이 높다.](#ox-6) **(O / X)**

> <a id="ox-1"></a>
> **1. 정답**: X
> **해설**: 파이썬 변수 이름은 영문자나 밑줄(`_`)로 시작해야 하며, 숫자로 시작할 수 없습니다. (예: `player1` (O), `1player` (X))

> <a id="ox-2"></a>
> **2. 정답**: X
> **해설**: 파이썬은 **동적 타이핑(Dynamic Typing)** 언어이므로, 변수에 값이 할당될 때 데이터 타입이 자동으로 결정됩니다. 타입을 미리 선언할 필요가 없습니다.

> <a id="ox-3"></a>
> **3. 정답**: X
> **해설**: 프로그래밍에서 등호(`=`)는 오른쪽의 값을 왼쪽의 변수에 **할당(assignment)**하라는 명령입니다. 수학의 '같다'와는 다른 의미입니다.

> <a id="ox-4"></a>
> **4. 정답**: O
> **해설**: 파이썬은 문자열을 표현할 때 작은따옴표(`'`)와 큰따옴표(`"`)를 구분 없이 사용할 수 있어 유연합니다. (예: `'hello'`, `"world"`)

> <a id="ox-5"></a>
> **5. 정답**: X
> **해설**: `print()` 함수에 쉼표(`,`)로 여러 값을 전달하면, 각 값 사이에 기본적으로 공백이 하나씩 추가되어 출력됩니다. 따라서 `Hello World`가 출력됩니다.

> <a id="ox-6"></a>
> **6. 정답**: O
> **해설**: 파이썬은 메모리 효율성을 위해 자주 사용되는 작은 정수(-5 ~ 256)나 짧은 문자열을 미리 만들어두고 재사용합니다. 따라서 `a`와 `b`는 같은 메모리 주소를 가리킬 가능성이 높습니다.
